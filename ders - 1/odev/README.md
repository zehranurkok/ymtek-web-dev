# Ödev
* Bir adet index.html adında bir dosya oluşturun.
* HTML iskelet yapısını yazın, ! ile hazır yapı kullanmayın.
* Sayfanın başlığı "Blue Screen of Death" olsun.
* Dosyanın (icerik.txt) içerisindeki yazıları görüntüdeki gibi sayfaya işleyin.
* Görüntüdeki sayfa şu özellikleri taşımaktadır:
    * arka plan rengi "#000066",
    * font rengi "white",
    * font büyüklüğü "5px",
    * font ailesi "Courier"

# Notlar
* Çoğu tarayıcıda web sayfalarını f11'e basarak tam ekran olarak görebilirsiniz.
* Çoğu tarayıcıda f12'ye basarak web sayfasının önyüzündeki kodları görebilirsiniz.
